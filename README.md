# استوديو الخيال 🎨 — STS Academy

أداة تخطيط إبداعي للأطفال **قبل** ما يبنوا مشروعهم في Scratch أو Scratch Jr.
موقع ثابت 100% — كل حاجة بتشتغل في المتصفح، مفيش سيرفر.

## رحلة الطفل
1. **يرسم** مشهده كله على ورقة (شخصيات + أشياء + خلفية)
2. **يصوّر** الرسمة ويرفعها — الموقع بيفرد الورقة لوحده
3. **يختار** نوع فكرته: قصة 📖 / أنيميشن 🎬 / لعبة 🎮 / مشهد تفاعلي ✨
4. **يقصّ عناصر** من رسمته (يحوّط عليها) وتبقى شخصيات تتحرك — والرسمة كلها بتفضل خلفية
5. **يجاوب أسئلة** ودّية عن فكرته في تاب "💡 أفكاري"
6. **يبني قواعد** ببطاقات مش بلوكات: [إمتى 🟧] + [مين] + [يعمل إيه 🟦]
7. **يجرّب فكرته** — العناصر بتتحرك وبتتكلم بفقاعات، فيه جمع نقط ولمس وكسب وخسارة
8. **يطلع خطة البناء** خطوة بخطوة لـ Scratch أو Scratch Jr — ويطبعها

## البطاقات
- **بطاقات "إمتى" 🟧:** في البداية / لما أدوس عليه / لما يلمس… / بعد كام ثانية / لما يوصل لمكان / لما النقط تكفي
- **بطاقات "يعمل إيه" 🟦:** يتحرك / يتكلم / يختفي / يظهر / يتجمع / خطر / مكسب / خسارة / يغيّر المشهد / صوت / يستنى / يتبع

## الملفات
- `index.html` — الواجهة كلها
- `studio.js` — معالجة الصورة + المشاهد المتعددة + محرك التجربة + مولّد خطط Scratch/ScratchJr

## الرفع على GitHub Pages
1. افتحي الريبو `sts-game` على GitHub وامسحي الملفات القديمة
2. Add file → Upload files → ارفعي `index.html` و `studio.js`
3. استني دقيقة وافتحي: https://phebronia.github.io/sts-game/

## ملاحظات للمدرّسين
- زرار ⭐ على أي عنصر يخليه "اللاعب" (بيتحرك بالأسهم / بأزرار اللمس على الموبايل)
- خطة Scratch بتشمل: الكائنات والخلفية والمتغيرات وخطوات كل قاعدة بالبلوكات المقترحة
- خطة Scratch Jr بتشمل: الصفحات والشخصيات وخطوات بلوكات جونيور (وبتنبّه لما حاجة مش موجودة في جونيور زي النقط)
- زرار 🖨️ بيطبع الخطة عشان الطفل ياخدها معاه وهو بيبني

## المشاهد المتعددة 🖼️
- تاب "🖼️ مشاهدي" فيه زرار ➕ لإضافة مشهد جديد (رسمة تانية) — من الكاميرا أو الجاليري
- بطاقة 🌙 «يغيّر المشهد» بتنقل لأي مشهد مرفوع (أو تحوّله ليل/غروب)
- في خطة Scratch كل مشهد = Backdrop، وفي Scratch Jr كل مشهد = صفحة

## تحديثات
- **أوامر لعناصر مختلفة في نفس القاعدة:** عند إضافة أي بطاقة زرقا بيسألك "مين اللي يعمل ده؟" — يعني ينفع: "لما أدوس على ١ → ١ يختفي، و«٢» يظهر". وفي خطة Scratch ده بيتترجم لـ«بث رسالة» 📨 والكائن التاني «عندما أتلقى»، وفي جونيور بالرسالة البرتقالية.
- **احتواء الصورة بنسبتها:** الصور الطويلة بتظهر بالطول كاملة من غير ضغط، والهوامش بتتملي بلون الورق.
- **كل مشهد بعناصره:** لما تشتغل على مشهد، مش بتشوف غير عناصره هو — في اللوحة وعلى المسرح وفي التجربة. وفي باني القواعد كل عنصر متعلّم باسم مشهده.
- **فقاعة كلام ذكية:** بتكبر وبتتقسم سطور على قد الكلام، ولو مفيش مكان فوق العنصر بتظهر تحته — ودايمًا جوه الشاشة.
- **الشخصية تنطق 🗣️:** جوه بطاقة «يتكلم» فيه اختيار «ينطق الكلام بصوت كمان» — بيستخدم صوت عربي من المتصفح. وفي خطة Scratch بيتترجم لإضافة «تحويل النص إلى كلام»، وفي جونيور لتسجيل الصوت بالميكروفون.
- **تحكم كيبورد وزراير شاشة:** بطاقة «⌨️ لما أدوس زرار» (المسافة/الأسهم) + بطاقة «🦘 ينط». على اللابتوب بالكيبورد، وعلى الموبايل الزراير المستخدمة بتظهر لوحدها على الشاشة وقت التجربة. في خطة Scratch بتتترجم لحدث «عند ضغط مفتاح»، وجونيور بينبّه إنه مفيهوش كيبورد ويقترح اللمس.
- **تعديل القواعد ✏️:** كل قاعدة جنبها زرار تعديل بيفتحها بكل اختياراتها عشان تغيّري فيها بدل المسح وإعادة الإنشاء.
- **مقاسات موبايل مظبوطة:** جوه الاستوديو الهيدر بيتضغط والمسرح بياخد عرض الشاشة والبطاقات والأزرار بمقاسات لمس مريحة.
- `mobile-preview.html` — معاينة مستقلة لمقاسات الموبايل بصورة تجريبية (مش جزء من الموقع، للمراجعة بس).
- **بوكس زراير اللعب:** الزراير بقت في بوكس منفصل تحت المسرح (مش فوق اللعبة) بحجم 52px، وبيظهر بس لو التجربة محتاجة زراير على جهاز لمس.
- **الكلام الطويل مبيطلعش بره:** جملة القاعدة والـ chips وقائمة القواعد وخطة البناء كلهم بيلفوا الكلام جواهم، وخانة «يقول إيه؟» ليها حد 140 حرف.
